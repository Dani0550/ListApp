In this project, which is a real challenge from Frontend Mentor, I have created an application that allows you to save content and create a dynamic list. This list can be manipulated in many ways, deleting the contents of the list, adding new items to the list and even filtering its contents.

But it's not only a frontend work in my case, as the challenge proposed, but I have delved into the backend, working with browser databases, indexedDB. In this way I put into practice my skills with databases, and to be able to manipulate the information they contain.
